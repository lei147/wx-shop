### WJ商城（微信小程序端）

- 界面参照某商城+蓝湖页面设计制作
- 测试数据采自京东商城
- 功能和数据库均有自己开发设计
- 服务端api基于node.js+mysql
- 计划添加基于vue-element-admin框架的后台管理

 **注意：当前版本功能还未完善，请勿商用。**
